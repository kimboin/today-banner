# 하루 1명 현수막 사이트

텍스트를 먼저 입력한 한 사람만 하루 동안 현수막을 선점할 수 있는 프로젝트입니다.
날짜가 바뀌면 자동으로 초기화되어 다시 선점 가능합니다.

## 기능

- 당일 최초 1회만 텍스트 등록 가능
- 등록 이후 당일에는 수정/재등록 불가
- 날짜 변경 시 자동 리셋
- 현재 시간 표시

## 실행 방법

```bash
cd today-banner
npm start
```

브라우저에서 `http://localhost:4000` 접속.

## 환경 변수

- `PORT`: 서버 포트 (기본값 `4000`)
- `HOST`: 서버 바인딩 주소 (기본값 `0.0.0.0`, 외부 접속 가능)

예시:

```bash
HOST=0.0.0.0 PORT=4000 npm start
```
